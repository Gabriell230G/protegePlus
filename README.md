# Protege+

Sistema digital de denúncias desenvolvido no projeto HackGov com o objetivo de facilitar a comunicação entre cidadãos e o poder público.

## 📌 Sobre o Projeto
O **Protege+** é um canal digital que permite registrar denúncias relacionadas a:
- violência doméstica
- abuso sexual
- assédio
- violência contra crianças ou idosos
- discriminação
- outras situações de risco

O sistema permite que cidadãos registrem ocorrências de forma simples, rápida e segura, inclusive de forma **anônima**.

## 🎯 Objetivo
Utilizar tecnologia para melhorar a comunicação entre população e governo, contribuindo para políticas públicas mais eficientes e baseadas em dados.

## ⚙️ Funcionalidades
- Formulário de denúncia
- Registro de denúncias anônimas
- Armazenamento das denúncias
- Dashboard administrativo
- Visualização de dados em gráficos
- Tabela com histórico de denúncias

## 📊 Dashboard
O painel administrativo apresenta:
- total de denúncias
- número de denúncias anônimas
- gráfico de denúncias por tipo
- gráfico de estados com mais denúncias
- tabela com registros completos

Para acessar o painel administrativo digite no "login: admin" e na "senha: 1234"

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Chart.js
- LocalStorage

## 📷 Demonstração
O sistema permite registrar denúncias e visualizar os dados em um painel de monitoramento.


## 📚 Projeto acadêmico
Este projeto foi desenvolvido como parte do **HackGov – Transformação Digital na Gestão Pública**, com o objetivo de propor soluções tecnológicas para melhorar serviços públicos.
