# 🛡️ Protege+

**`Sistema digital de denúncias voltado à proteção do cidadão`**

O **Protege+** é uma solução desenvolvida no projeto **HackGov**, com o objetivo de facilitar a comunicação entre cidadãos e o poder público por meio de um canal digital de denúncias.

A proposta é tornar o processo **mais acessível, rápido e seguro**, contribuindo para uma sociedade mais protegida e informada.

---

## 📌 Sobre o Projeto

O **Protege+** permite o registro de denúncias relacionadas a:

➽ Violência doméstica  
➽ Abuso sexual  
➽ Assédio  
➽ Violência contra crianças ou idosos  
➽ Discriminação  
➽ Outras situações de risco  

O sistema foi pensado para garantir **facilidade de uso e segurança**, permitindo inclusive o envio de denúncias de forma **anônima**.

---

## 🎯 Objetivo

Utilizar a tecnologia como ferramenta de impacto social, melhorando a comunicação entre população e governo e contribuindo para a criação de **políticas públicas mais eficientes e baseadas em dados reais**.

---

## ⚙️ Funcionalidades

➽ Formulário intuitivo para registro de denúncias  
➽ Opção de envio anônimo  
➽ Armazenamento local das informações  
➽ Dashboard administrativo  
➽ Visualização de dados em gráficos  
➽ Tabela com histórico completo de denúncias  

---

## 📊 Dashboard

O painel administrativo permite acompanhar os dados de forma clara e estratégica:

➽ Total de denúncias registradas  
➽ Quantidade de denúncias anônimas  
➽ Gráfico de denúncias por tipo  
➽ Gráfico de estados com maior incidência  
➽ Tabela com todos os registros  

🔐 **Acesso ao painel:**

➽ Login: `admin`  
➽ Senha: `1234`
